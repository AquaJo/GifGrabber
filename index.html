Giphy-Extractor


Don't want to register for an api key by giphy for filtering max. 50 gifs of a keyword (beta-key) (I think). Just want to extract some urls of these gifs more automated, not for more complex api stuff. You might be at the right place. Extract Giphy Images relatively easy with this tool.


For them prefering light mode, press 'l'.

Might create a webscraper-service to make following steps superfluous in the future.


How To Use?

Go to Giphy and search for a topic or go to any giphy-page with gifs you like.

Then open dev console with Strg+Shift+I. The trick is to copy the html of the page, then extract from it. You can scroll down the page to get even more gifs, but keep in mind the data quickly gets 'too' large for the input on this page. When you are finished copy the html.

Now paste this html data into this text-area and click "get image-links".
