Giphy-Extractor


Tired of registering for a restrictive, yet free beta API key from Giphy? Want to extract URLs more efficiently or download multiple GIFs at once without the hassle of complex API functions? Look no further. This tool makes it easy to extract many Giphy images with just a few simple steps.


If you prefer light mode, press 'l'.

I might create a serverside webscraper-service to make following steps superfluous in the future.


How To Use?

Visit Giphy, search for a topic or navigate to a page with GIFs you like and scroll down the page to load as many gifs you want.

Once finished, open the developer console by pressing Ctrl+Shift+I.

Click the html-tag in the Elements tab and press Ctrl+C to copy.

Please keep in mind that some browsers might have different ways to access developer console, it's best to check the browser documentation for the same.

Paste the HTML into the provided textarea and press "get image-links".
